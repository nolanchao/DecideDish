# Where2Eat
# www.where2eat.me

##How to use
###### Download (of course)
###### Obtain API Key from Yelp via https://www.yelp.com/developers/documentation/v3/get_started
###### Plug said key into views.py
###### Run sqlite3 app.db < schema.sql command
###### > sqlite3 app.db < schema.sql
###### Run command
###### > python run.py
